SDHпјҢдёҖеӯҰе°ұдјҡ

еҸ‘иЎЁдәҺ 2021-05-19 еҲҶзұ»дәҺ жңәеҷЁдәәжҺ§еҲ¶ пјҢ дё“ж ҸвҖ”вҖ”жңәеҷЁдәәеҠЁеҠӣеӯҰ йҳ…иҜ»ж¬Ўж•°пјҡ Valineпјҡ
жң¬ж–Үеӯ—ж•°пјҡ 6.7k йҳ…иҜ»ж—¶й•ҝ ≈ 6 еҲҶй’ҹ

д»Ӣз»Қж ҮеҮҶ DH(SDH) жңәеҷЁдәәе»әзі»ж–№жі•гҖӮ

1 еүҚиЁҖ

еңЁеҒҡжңәеҷЁдәәжҺ§еҲ¶зҡ„ж—¶еҖҷпјҢдёәдәҶиғҪеӨҹж–№дҫҝең°еҜ№жңәеҷЁдәәиҝӣиЎҢеҠӣзҹ©жҺ§еҲ¶пјҢиҮӘе·ұжҗӯе»әдәҶдёҖеҘ—жҺ§еҲ¶еҷЁгҖӮеңЁжҗӯе»әе®Ңеҹәжң¬зҡ„зЎ¬д»¶е№іеҸ°еҗҺпјҢйңҖиҰҒеҜ№жңәеҷЁдәәиҝӣиЎҢе»әжЁЎгҖӮйҰ–е…ҲжҳҜжңәеҷЁдәәеқҗж Үзі»зҡ„е»әз«ӢгҖӮд»ҘUR10дёәдҫӢгҖӮ

жңүдёӨз§ҚеҸҜд»ҘеңЁ MATLAB дёӯдҪҝз”Ёзҡ„е·Ҙе…·з®ұпјҡ

  • PerterCokeзҡ„Robotics ToolBox
  • ARTE: A ROBOTICS TOOLBOX FOR EDUCATION

з¬¬дёҖз§Қе·Ҙе…·з®ұеҜ№еә”зҡ„жңүдёҖжң¬д№Ұ гҖҠRobotics, Vision and ControlгҖӢ[1]пјҢиҝҷдёӘз”Ёзҡ„жҜ”иҫғеӨҡпјҢдёҚеҶҚиөҳиҝ°гҖӮ

з¬¬дәҢз§Қе·Ҙе…·з®ұдёӢиҪҪеҘҪеҗҺе°Ҷи·Ҝеҫ„ж·»еҠ еҲ° MATLAB дёӯпјҢз„¶еҗҺеңЁе®ғзҡ„ж–Үд»¶еӨ№дёӢиҝҗиЎҢдёҖдёӢе‘Ҫд»ӨеҚіеҸҜгҖӮ

2 SDHе»әз«Ӣеқҗж Үзі»зҡ„иҝҮзЁӢ

дёҠеӣҫжүҖзӨәзҡ„иҪ¬еҠЁе…іиҠӮ DH еҸӮж•°зҡ„е®ҡд№ү[1]пјҡ

  1. е…іиҠӮи·қзҰ» $d_j$пјҡд»Һ $x_{j-1}$ иҪҙжІҝ $z_{j-1}$ иҪҙеҲ° $x_j$ иҪҙзҡ„и·қзҰ»пјӣ
  2. е…іиҠӮиҪ¬и§’ $\theta _j$пјҡд»Һ $x_{j-1}$ иҪҙжІҝ $z_{j-1}$ иҪҙиҪ¬еҲ° $x_j$ иҪҙзҡ„и§’еәҰпјҢдё” $\theta_j \in (-\pi,\pi]$пјӣ
  3. жқҶй—ҙи·қзҰ» $a_j$пјҡд»Һ $z_{j-1}$ иҪҙжІҝ $x_j$ иҪҙеҲ° $z_j$ иҪҙзҡ„и·қзҰ»пјӣ
  4. жқҶд»¶жүӯи§’ $\alpha_j$пјҡд»Һ $z_{j-1}$ иҪҙжІҝ $x_j$ иҪҙиҪ¬еҲ° $z_j$ иҪҙзҡ„и§’еәҰпјҢдё” $\alpha_j \in (-\pi,\pi]$.

DH еқҗж Үзі»дёӯиҪҙдёҺеҺҹзӮ№зЎ®е®ҡзҡ„еҹәжң¬еҺҹеҲҷжҳҜ[2]пјҡ

  1. $z_j$ иҪҙжІҝе…іиҠӮ $j+1$ зҡ„иҪҙеҗ‘(жҢҮеҗ‘еҸҜд»»йҖүпјҢдҪҶйҖҡеёёйғҪе°Ҷеҗ„е№іиЎҢзҡ„ $z$ иҪҙеқҮеҸ–дёәзӣёеҗҢзҡ„жҢҮеҗ‘)пјӣ
  2. еҺҹзӮ№йҖүеҸ–зҡ„еҺҹеҲҷпјҡеҺҹзӮ№ $O_j$ дёә $z_{j-1}$ иҪҙе’Ң $z_j$ иҪҙзҡ„е…¬жі•зәҝдёҺ $z_{j-1}$ иҪҙзҡ„дәӨзӮ№.
    • еҪ“ $z_{j-1}$ иҪҙе’Ң $z_j$ иҪҙе№іиЎҢж—¶пјҢдёӨиҪҙд№Ӣй—ҙжңүж— ж•°жқЎе…¬еһӮзәҝпјҢиӢҘдёӨиҪҙйҮҚеҗҲпјҢеҸ– $O_j = O_{j-1}$пјӣиӢҘдёӨиҪҙе№іиЎҢдҪҶдёҚйҮҚеҗҲпјҢеҲҷиҝҮ $O_{j-1}$ зӮ№дҪң $z_{j-1}$ иҪҙе’Ң $z_j$ иҪҙзҡ„е…¬еһӮзәҝпјҢжӯӨе…¬еһӮзәҝдёҺ $z_j$ иҪҙзҡ„дәӨзӮ№еҚі $O_j$.
    • з”ұдәҺжІЎжңү $z_{-1}$пјҢжүҖд»ҘдёҠиҝ°ж–№жі•дёҚйҖӮз”ЁдәҺ $O_0$пјҢиҝҷж—¶зЎ®е®ҡ $O_0$ зҡ„ж–№жі•жҳҜпјҡиӢҘ $z_0$ дёҺ $z_1$ зӣёдәӨпјҢеҸ– $O_0 = O_1$пјӣиӢҘ$z_0$ дёҺ $z_1$ дёҚзӣёдәӨпјҢеҸ– $O_0$ еңЁ $z_0$ дёҺ $z_1$ зҡ„е…¬еһӮзәҝдёҠгҖӮ
  3. иҪҙ $x_j$ дёҺиҪҙ $z_{j-1}$ зӣёдәӨдё”еһӮзӣҙпјҢеҚі $x_j$ дёәжІҝиҝҮ $z_{j-1}$ иҪҙе’Ң $z_j$ иҪҙзҡ„е…¬еһӮзәҝпјҢж–№еҗ‘д»Һ $z_{j-1}$ жҢҮеҗ‘ $z_j$.
    • еҪ“ $z_{j-1}$ иҪҙе’Ң $z_j$ иҪҙйҮҚеҗҲж—¶пјҢиҝҷж—¶ $O_j = O_{j-1}$пјҢйӮЈд№ҲеңЁйӣ¶дҪҚж—¶иҰҒжұӮ $x_j = x_{j-1}$.
    • еҪ“ $z_{j-1}$ иҪҙе’Ң $z_j$ иҪҙзӣёдәӨдё”дёҚйҮҚеҗҲж—¶пјҢйҖүжӢ© $x_j = \pm(z_{j-1}\times z_j)$пјҢйҖҡеёёдҪҝжүҖжңүе№іиЎҢзҡ„ $x$ иҪҙеқҮе…·жңүзӣёеҗҢзҡ„жҢҮеҗ‘.
    • еҪ“ $j = 0$ж—¶пјҢ$O_0 = O_1$, жҲ– $O_0$ еңЁ $z_0$ иҪҙе’Ң $z_1$ иҪҙзҡ„е…¬еһӮзәҝдёҠпјҢйҖүеҸ– $x_0$ иҪҙдёҺ $x_1$ иҪҙйҮҚеҗҲ.

      Tip: дёҠиҝ°ж–№жі•жІЎжңүиҖғиҷ‘е№іеҠЁе…іиҠӮзҡ„е»әзі»пјҢеҸҜеҸӮи§Ғ[2]гҖӮ

3 е®ҳзҪ‘з»ҷеҮәзҡ„ UR10 зҡ„ SDH жЁЎеһӢ

URе®ҳзҪ‘з»ҷеҮәеңЁйӣ¶дҪҚж—¶зҡ„UR10жЁЎеһӢеҰӮдёӢеӣҫжүҖзӨәгҖӮ

жҢүз…§з¬¬ 2 иҠӮз»ҷеҮәзҡ„ DH е»әзі»еҺҹеҲҷпјҢиҝӣиЎҢд»ҘдёӢжӯҘйӘӨзҡ„е»әжЁЎпјҡ

  1. зЎ®е®ҡеҗ„е…іиҠӮзҡ„ж—ӢиҪ¬иҪҙпјҢз”Ё $q_j$ иЎЁзӨә(еҸҜд»ҘеҸӮиҖғдёӢдёҖиҠӮзҡ„еӣҫ)гҖӮ
  2. зЎ®е®ҡ $z$ иҪҙпјҢдҫқжҚ® DH е»әзі»ж–№жі•пјҢ $z_0 (z_{base})$ еә”дёә $q_1$пјҢ$z_1$ еә”дёә $q_2$пјҢд»ҘжӯӨзұ»жҺЁпјҢжңҖз»ҲпјҢ$z_6$ еә”дёә $q_6$пјҢдёҺ $z_5$ зӣёеҗҢгҖӮ
  3. зЎ®е®ҡеқҗж ҮеҺҹзӮ№пјҡ
    • $j = 0$ ж—¶пјҢ$z_0$ дёҺ $z_1$ зӣёдәӨпјҢеҸ– $O_0 = O_1$пјӣ
    • $j = 1$ ж—¶пјҢ$z_0$ дёҺ $z_1$ зӣёдәӨпјҢеҸ–е…¶дәӨзӮ№дёә $O_1$пјӣ
    • $j = 2$ ж—¶пјҢ$z_1$ дёҺ $z_2$ е№іиЎҢпјҢиҝҮ $O_1$ дҪң $z_1$ дёҺ $z_2$ зҡ„е…¬еһӮзәҝе№¶дёҺ $z_2$ зӣёдәӨзҡ„дәӨзӮ№дёә $O_2$пјӣ
    • $j = 3$ ж—¶пјҢ$z_2$ дёҺ $z_3$ е№іиЎҢпјҢиҝҮ $O_2$ дҪң $z_2$ дёҺ $z_3$ зҡ„е…¬еһӮзәҝе№¶дёҺ $z_3$ зӣёдәӨзҡ„дәӨзӮ№дёә $O_3$пјӣ
    • $j = 4$ ж—¶пјҢ$z_3$ дёҺ $z_4$ зӣёдәӨпјҢеҸ–е…¶дәӨзӮ№дёә $O_4$пјӣ
    • $j = 5$ ж—¶пјҢ$z_4$ дёҺ $z_5$ зӣёдәӨпјҢеҸ–е…¶дәӨзӮ№дёә $O_5$пјӣ
    • $j = 6$ ж—¶пјҢ$z_5$ дёҺ $z_6$ йҮҚеҗҲпјҢеҸ–еҺҹзӮ№дёә $O_6 = O_5$пјӣ
  4. зЎ®е®ҡ $x$ иҪҙпјҡ
    • $j = 0$ ж—¶пјҢ$O_0 = O_1$пјҢеә”жңү $x_0 = x_1$пјӣ
    • $j = 1$ ж—¶пјҢ$z_0$ дёҺ $z_1$ зӣёдәӨпјҢйҖүжӢ© $x_1 = z_0 \times z_1$ (д»ҘдёҠеӣҫдёәдҫӢпјҢеҸҜд»Ҙи®ӨдёәжҳҜеӣҫеҶ…жҢҮеҗ‘еҗ‘еҸі)пјӣ
    • $j = 2$ ж—¶пјҢ$z_1$ дёҺ $z_2$ е№іиЎҢпјҢ$x_2$ еҚі $z_1$ дёҺ $z_2$ зҡ„е…¬еһӮзәҝпјҢдё”д»Һ $z_1$ жҢҮеҗ‘ $z_2$пјҢдёәдәҶдҝқжҢҒе№іиЎҢж–№еҗ‘зҡ„ $x$ иҪҙжңқеҗ‘дёҖиҮҙпјҢжҢҮе®ҡ $x_2$ д»Һ $z_2$ жҢҮеҗ‘ $z_1$пјӣ
    • $j = 3$ ж—¶пјҢ$z_2$ дёҺ $z_3$ е№іиЎҢпјҢ$x_3$ еҚі $z_2$ дёҺ $z_3$ зҡ„е…¬еһӮзәҝпјҢдё”д»Һ $z_2$ жҢҮеҗ‘ $z_3$пјҢдёәдәҶдҝқжҢҒе№іиЎҢж–№еҗ‘зҡ„ $x$ иҪҙжңқеҗ‘дёҖиҮҙпјҢжҢҮе®ҡ $x_3$ д»Һ $z_3$ жҢҮеҗ‘ $z_2$пјӣ
    • $j = 4$ ж—¶пјҢ$z_3$ дёҺ $z_4$ зӣёдәӨпјҢйҖүжӢ© $x_4 = z_3 \times z_4$пјӣ
    • $j = 5$ ж—¶пјҢ$z_4$ дёҺ $z_5$ зӣёдәӨпјҢйҖүжӢ© $x_5 = z_4 \times z_5$пјҢдёәдәҶдҝқжҢҒе№іиЎҢж–№еҗ‘зҡ„ $x$ иҪҙжңқеҗ‘дёҖиҮҙпјҢж”№дёә$x_5 = -z_4 \times z_5$пјӣ
    • $j = 6$ ж—¶пјҢ$z_5$ дёҺ $z_6$ йҮҚеҗҲпјҢеҸ– $x_6 = x_5$пјӣ

дёҠиҝ°е»әжЁЎеҗҺеӯҳеңЁзҡ„й—®йўҳпјҡ

  1. $0$ зі»е’Ң $1$ зі»йҮҚеҗҲпјҢиҖҢ $O_1$ дёә $z_0$ дёҺ $z_1$ зҡ„дәӨзӮ№пјҢ иҝҷж ·жІЎжңүиҖғиҷ‘еҲ° $O_1$ дёҺең°йқўзҡ„й«ҳеәҰпјҢеӣ жӯӨпјҢе°Ҷ $O_0$ з§»еҠЁиҮіеә•еә§дёҺең°йқўзҡ„жҺҘи§ҰйқўдёҠпјҢеқҗж Үзі»ж–№еҗ‘дёҚеҸҳпјӣ
  2. еҗҢзҗҶпјҢ$5$ зі»е’Ң $6$ зі»йҮҚеҗҲпјҢжІЎжңүиҖғиҷ‘еҲ°иҝһжқҶ $6$ зҡ„й•ҝеәҰпјҢеӣ жӯӨе°Ҷ $O_6$ з§»еҠЁиҮіжңәеҷЁдәәжң«з«ҜпјҢеқҗж Үзі»жңқеҗ‘дёҚеҸҳгҖӮ

зЎ®е®ҡ DH еҸӮж•°пјҡ

  1. $j = 1$ ж—¶пјҡ
    • д»Һ $x_0$ иҪҙжІҝ $z_0$ иҪҙеҲ° $x_1$ иҪҙзҡ„и·қзҰ»дёә $d_1$пјҢеҚі $O_1$ еҲ°ең°йқўзҡ„й«ҳеәҰпјӣ
    • еӣ  $x_0 // x_1$пјҢд»Һ $x_0$ иҪҙжІҝ $z_0$ иҪҙиҪ¬еҲ° $x_1$ иҪҙзҡ„и§’еәҰдёә $\theta_1 = 0$пјӣ
    • еӣ  $z_0 \perp z_1$ дё”е…ұйқўпјҢд»Һ $z_0$ иҪҙжІҝ $x_1$ иҪҙеҲ° $z_1$ иҪҙзҡ„и·қзҰ»дёә $a_1 = 0$пјӣ
    • д»Һ $z_0$ иҪҙжІҝ $x_1$ иҪҙиҪ¬еҲ° $z_1$ иҪҙзҡ„и§’еәҰдёә $\alpha_1 = \pi/2$.
  2. $j = 2$ ж—¶пјҡ
    • д»Һ $x_1$ иҪҙжІҝ $z_1$ иҪҙеҲ° $x_2$ иҪҙзҡ„и·қзҰ»дёә $d_2 = 0$пјӣ
    • д»Һ $x_1$ иҪҙжІҝ $z_1$ иҪҙиҪ¬еҲ° $x_2$ иҪҙзҡ„и§’еәҰдёә $\theta_2 = 0$пјӣ
    • д»Һ $z_1$ иҪҙжІҝ $x_2$ иҪҙеҲ° $z_2$ иҪҙзҡ„и·қзҰ»дёә $-a_2$пјҢеҚі иҝһжқҶ $2$ й•ҝеәҰзҡ„зӣёеҸҚж•°пјӣ
    • еӣ  $z_1 // z_2$пјҢд»Һ $z_1$ иҪҙжІҝ $x_2$ иҪҙиҪ¬еҲ° $z_2$ иҪҙзҡ„и§’еәҰдёә $\alpha_2 = 0$.
  3. $j = 3$ ж—¶пјҡ
    • д»Һ $x_2$ иҪҙжІҝ $z_2$ иҪҙеҲ° $x_3$ иҪҙзҡ„и·қзҰ»дёә $d_3 = 0$пјӣ
    • еӣ  $x_2 // x_3$пјҢд»Һ $x_2$ иҪҙжІҝ $z_2$ иҪҙиҪ¬еҲ° $x_3$ иҪҙзҡ„и§’еәҰдёә $\theta_3 = 0$пјӣ
    • д»Һ $z_2$ иҪҙжІҝ $x_3$ иҪҙеҲ° $z_3$ иҪҙзҡ„и·қзҰ»дёә $-a_3$пјҢеҚі иҝһжқҶ $3$ з«–зӣҙж®өзҡ„й•ҝеәҰпјӣ
    • еӣ  $z_2 // z_3$пјҢд»Һ $z_2$ иҪҙжІҝ $x_3$ иҪҙиҪ¬еҲ° $z_3$ иҪҙзҡ„и§’еәҰдёә $\alpha_3 = 0$.
  4. $j = 4$ ж—¶пјҢ
    • д»Һ $x_3$ иҪҙжІҝ $z_3$ иҪҙеҲ° $x_4$ иҪҙзҡ„и·қзҰ»дёә $d_4$пјҢеҚіиҝһжқҶ $4$ й•ҝеәҰпјӣ
    • еӣ  $x_3 // x_4$ пјҢд»Һ $x_3$ иҪҙжІҝ $z_3$ иҪҙиҪ¬еҲ° $x_4$ иҪҙзҡ„и§’еәҰдёә $\theta_4 = 0$пјӣ
    • еӣ  $z_3 \perp z_4$ дё”е…ұйқўпјҢд»Һ $z_3$ иҪҙжІҝ $x_4$ иҪҙеҲ° $z_4$ иҪҙзҡ„и·қзҰ»дёә $a_4 = 0$пјӣ
    • д»Һ $z_3$ иҪҙжІҝ $x_4$ иҪҙиҪ¬еҲ° $z_4$ иҪҙзҡ„и§’еәҰдёә $\alpha_4 = \pi/2$.
  5. $j = 5$ ж—¶пјҢ
    • д»Һ $x_4$ иҪҙжІҝ $z_4$ иҪҙеҲ° $x_5$ иҪҙзҡ„и·қзҰ»дёә $d_5$пјҢеҚіиҝһжқҶ $5$ й•ҝеәҰпјӣ
    • еӣ  $x_4 // x_5$пјҢд»Һ $x_4$ иҪҙжІҝ $z_4$ иҪҙиҪ¬еҲ° $x_5$ иҪҙзҡ„и§’еәҰдёә $\theta_5 = 0$пјӣ
    • еӣ  $z_4 \perp z_5$ дё”е…ұйқўпјҢд»Һ $z_4$ иҪҙжІҝ $x_5$ иҪҙеҲ° $z_5$ иҪҙзҡ„и·қзҰ»дёә $a_5 = 0$пјӣ
    • д»Һ $z_4$ иҪҙжІҝ $x_5$ иҪҙиҪ¬еҲ° $z_5$ иҪҙзҡ„и§’еәҰдёә $\alpha_5 = -\pi/2$.
  6. $j = 6$ ж—¶пјҢ
    • д»Һ $x_5$ иҪҙжІҝ $z_5$ иҪҙеҲ° $x_6$ иҪҙзҡ„и·қзҰ»дёә $d_6$пјҢеҚіиҝһжқҶ $6$ й•ҝеәҰпјӣ
    • еӣ  $x_5 // x_6$пјҢд»Һ $x_5$ иҪҙжІҝ $z_5$ иҪҙиҪ¬еҲ° $x_6$ иҪҙзҡ„и§’еәҰдёә $\theta_6 = 0$пјӣ
    • еӣ  $z_5$ дёҺ $z_6$ йҮҚеҗҲпјҢд»Һ $z_5$ иҪҙжІҝ $x_6$ иҪҙеҲ° $z_6$ иҪҙзҡ„и·қзҰ»дёә $a_6 = 0$пјӣ
    • д»Һ $z_5$ иҪҙжІҝ $x_6$ иҪҙиҪ¬еҲ° $z_6$ иҪҙзҡ„и§’еәҰдёә $\alpha_6 = 0$.

еҜ№з…§е®ҳзҪ‘з»ҷеҮәзҡ„ DH еҸӮж•°пјҡ

дёҠиҝ°з»ҷеҮәзҡ„ DH е»әзі»ж–№жі•еӯҳеңЁзҡ„дёҚи¶іпјҡ

  1. $j = 2$ ж—¶пјҢ$z_1$ дёҺ $z_2$ е№іиЎҢпјҢжҢүз…§еҹәжң¬еҺҹеҲҷеә”иҝҮ $O_1$ дҪң $z_1$ дёҺ $z_2$ зҡ„е…¬еһӮзәҝпјҢе…¶дёҺ $z_2$ зҡ„дәӨзӮ№дёә $O_2$гҖӮиҝҷж ·еӯҗе»әзі»еӯҳеңЁзҡ„й—®йўҳжҳҜжңӘе°ҶиҝһжқҶ $1$ зҡ„й•ҝеәҰиҖғиҷ‘еңЁеҶ…гҖӮ
  2. еҗҢзҗҶпјҢжңӘиҖғиҷ‘иҝһжқҶ $3$ еңЁ $z_2(q_3)$ иҪҙж–№еҗ‘зҡ„й•ҝеәҰгҖӮ
  3. е®ҳзҪ‘зҡ„йӣ¶дҪҚеңЁе®һйҷ…дёӯжҳҜиәәеңЁең°дёҠзҡ„дёҖз§ҚдҪҚе§ҝпјҢз¬”иҖ…еңЁе®һйҷ…еә”з”ЁдёӯиҰҒжұӮйӣ¶дҪҚжҳҜз«–зӣҙзҠ¶жҖҒгҖӮ
  4. д»ҝзңҹдёӯиҝӣиЎҢжҳҫзӨәзҡ„ж—¶еҖҷдјҡж„ҹи§үж•ҲжһңдёҚеҘҪгҖӮ

4 еҢ…еҗ«е…ЁйғЁеҸӮж•°зҡ„SDHе»әзі»ж–№жі•

йҰ–е…ҲпјҢе°Ҷйӣ¶дҪҚжҢҮе®ҡдёәз«–зӣҙзҠ¶жҖҒпјҢеҰӮдёӢеӣҫжүҖзӨәгҖӮ

еҶҚиҝӣиЎҢдёҖж¬Ўеқҗж Үзі»зҡ„зЎ®е®ҡпјҡ

  1. зЎ®е®ҡеҗ„е…іиҠӮзҡ„ж—ӢиҪ¬иҪҙпјҢз”Ё $q_j$ иЎЁзӨәгҖӮ
  2. зЎ®е®ҡ $z$ иҪҙпјҢдҫқжҚ® DH е»әзі»ж–№жі•пјҢ $z_0$ еә”дёә $q_1$пјҢ$z_1$ еә”дёә $q_2$пјҢд»ҘжӯӨзұ»жҺЁпјҢжңҖз»ҲпјҢ$z_6$ еә”дёә $q_6$пјҢдёҺ $z_5$ зӣёеҗҢгҖӮжҢҮе®ҡз«–зӣҙж–№еҗ‘зҡ„ $z$ иҪҙеҗ‘дёҠдёәжӯЈпјҢж°ҙе№іж–№еҗ‘зҡ„ $z$ иҪҙеҗ‘е·ҰдёәжӯЈгҖӮ
  3. зЎ®е®ҡеқҗж ҮеҺҹзӮ№пјҡ
    • $j = 0$ ж—¶пјҢдёәдәҶеҢ…еҗ« $1$ е…іиҠӮеҲ°ең°йқўзҡ„и·қзҰ»пјҢеә”еҸ–еҺҹзӮ№ $O_0$ дёәеҹәеә§дёҺең°йқўзҡ„жҺҘи§Ұеә•йқўдёӯеҝғпјӣ
    • $j = 1$ ж—¶пјҢ$z_0$ дёҺ $z_1$ зӣёдәӨпјҢеҸ–е…¶дәӨзӮ№дёә $O_1$пјҢдёҺз¬¬3иҠӮдёҖиҮҙпјӣ
    • $j = 2$ ж—¶пјҢ$z_1$ дёҺ $z_2$ е№іиЎҢпјҢз¬¬3иҠӮдёӯеҜ№еә”зҡ„дҪҚзҪ®дёә $O_2вҖҷ$пјҢдёәеҢ…еҗ«иҝһжқҶ $1$ зҡ„й•ҝеәҰпјҢе°ҶеҺҹжқҘзҡ„ $O_2$ з§»еҠЁиҮіиҝһжқҶ $2$ зҡ„дёӯеҝғзәҝдёҺ $z_2$ зҡ„дәӨзӮ№пјӣ
    • $j = 3$ ж—¶пјҢ$z_2$ дёҺ $z_3$ е№іиЎҢпјҢжҢүз…§еҹәжң¬еҺҹеҲҷеә”зЎ®е®ҡдёә $O_3вҖҷ$пјҢжіЁж„ҸеҲ°иҝһжқҶ $3$ е®һйҷ…дёҠз”ұз”өжңәжүҖеңЁж°ҙе№іж®өдёҺеҢ…еҗ« $4$ е…іиҠӮеңЁеҶ…зҡ„з«–зӣҙж®өз»„жҲҗпјҢдёәеҢ…еҗ«иҝһжқҶ $3$ еңЁж°ҙе№іж®өзҡ„й•ҝеәҰпјҢеә”иҜҘз§»иҮіиҝһжқҶ $3$ дёҺ $z_3$ зҡ„дәӨзӮ№пјҢеҚі $O_3$пјҢеҸҚиҖҢдёҺз¬¬3иҠӮзҡ„ $O_3$ дёҖиҮҙпјӣ
    • $j = 4$ ж—¶пјҢ$z_3$ дёҺ $z_4$ зӣёдәӨпјҢеҸ–е…¶дәӨзӮ№дёә $O_4$пјҢдёҺз¬¬3иҠӮдёҖиҮҙпјӣ
    • $j = 5$ ж—¶пјҢ$z_4$ дёҺ $z_5$ зӣёдәӨпјҢеҸ–е…¶дәӨзӮ№дёә $O_5$пјҢдёҺз¬¬3иҠӮдёҖиҮҙпјӣ
    • $j = 6$ ж—¶пјҢ$z_5$ дёҺ $z_6$ йҮҚеҗҲпјҢдёәеҢ…еҗ«иҝһжқҶ $6$ зҡ„й•ҝеәҰпјҢеҸ–еҺҹзӮ№дёә $O_6$пјҢеңЁжң«з«Ҝпјӣ
  4. зЎ®е®ҡ $x$ иҪҙпјҡ
    • $j = 0$ ж—¶пјҢдёҺз¬¬3иҠӮдёҖиҮҙпјӣ
    • $j = 1$ ж—¶пјҢ$z_0$ дёҺ $z_1$ зӣёдәӨпјҢйҖүжӢ© $x_1 = z_0 \times z_1$ (д»ҘдёҠеӣҫдёәдҫӢпјҢеҸҜд»Ҙи®ӨдёәжҳҜжҢҮеҗ‘зәёйқўеӨ–)пјӣ
    • $j = 2$ ж—¶пјҢ$z_1$ дёҺ $z_2$ е№іиЎҢпјҢ$x_2$ еҚі $z_1$ дёҺ $z_2$ зҡ„е…¬еһӮзәҝпјҢдё”д»Һ $z_1$ жҢҮеҗ‘ $z_2$пјӣ
    • $j = 3$ ж—¶пјҢ$z_2$ дёҺ $z_3$ е№іиЎҢпјҢ$x_3$ еҚі $z_2$ дёҺ $z_3$ зҡ„е…¬еһӮзәҝпјҢдё”д»Һ $z_2$ жҢҮеҗ‘ $z_3$пјӣ
    • $j = 4$ ж—¶пјҢ$z_3$ дёҺ $z_4$ зӣёдәӨпјҢйҖүжӢ© $x_4 = -z_3 \times z_4$пјҢдёҺ $x_1$ дҝқжҢҒж–№еҗ‘дёҖиҮҙпјӣ
    • $j = 5$ ж—¶пјҢ$z_4$ дёҺ $z_5$ зӣёдәӨпјҢйҖүжӢ© $x_5 = z_4 \times z_5$пјӣ
    • $j = 6$ ж—¶пјҢ$z_5$ дёҺ $z_6$ йҮҚеҗҲпјҢеҸ– $x_6 // x_5$ дё”жӯЈж–№еҗ‘зӣёеҗҢпјӣ

зЎ®е®ҡ DH еҸӮж•°пјҡ

  1. $j = 1$ ж—¶пјҡ
    • д»Һ $x_0$ иҪҙжІҝ $z_0$ иҪҙеҲ° $x_1$ иҪҙзҡ„и·қзҰ»дёә $d_1$пјҢеҚі $O_1$ еҲ°ең°йқўзҡ„й«ҳеәҰпјӣ
    • еӣ  $x_0 // x_1$пјҢд»Һ $x_0$ иҪҙжІҝ $z_0$ иҪҙиҪ¬еҲ° $x_1$ иҪҙзҡ„и§’еәҰдёә $\theta_1 = 0$пјӣ
    • еӣ  $z_0 \perp z_1$ дё”е…ұйқўпјҢд»Һ $z_0$ иҪҙжІҝ $x_1$ иҪҙеҲ° $z_1$ иҪҙзҡ„и·қзҰ»дёә $a_1 = 0$пјӣ
    • д»Һ $z_0$ иҪҙжІҝ $x_1$ иҪҙиҪ¬еҲ° $z_1$ иҪҙзҡ„и§’еәҰдёә $\alpha_1 = \pi/2$.
  2. $j = 2$ ж—¶пјҡ
    • д»Һ $x_1$ иҪҙжІҝ $z_1$ иҪҙеҲ° $x_2$ иҪҙзҡ„и·қзҰ»дёә $d_2$пјҢеҚі $z_0$ дёҺиҝһжқҶ $2$ дёӯиҪҙзәҝзҡ„и·қзҰ»пјҢеҚіиҝһжқҶ $1$ зҡ„й•ҝеәҰпјӣ
    • д»Һ $x_1$ иҪҙжІҝ $z_1$ иҪҙиҪ¬еҲ° $x_2$ иҪҙзҡ„и§’еәҰдёә $\theta_2 = \pi/2$пјӣ
    • д»Һ $z_1$ иҪҙжІҝ $x_2$ иҪҙеҲ° $z_2$ иҪҙзҡ„и·қзҰ»дёә $a_2$пјҢеҚі иҝһжқҶ $2$ зҡ„й•ҝеәҰпјӣ
    • еӣ  $z_1 // z_2$пјҢд»Һ $z_1$ иҪҙжІҝ $x_2$ иҪҙиҪ¬еҲ° $z_2$ иҪҙзҡ„и§’еәҰдёә $\alpha_2 = 0$.
  3. $j = 3$ ж—¶пјҢжіЁж„ҸиҝһжқҶ $3$ е®һйҷ…дёҠз”ұз”өжңәжүҖеңЁж°ҙе№іж®өдёҺеҢ…еҗ« $4$ е…іиҠӮеңЁеҶ…зҡ„з«–зӣҙж®өз»„жҲҗпјҡ
    • д»Һ $x_2$ иҪҙжІҝ $z_2$ иҪҙеҲ° $x_3$ иҪҙзҡ„и·қзҰ»дёә $-d_3$пјҢеҚіиҝһжқҶ $3$ ж°ҙе№іж®өй•ҝеәҰзҡ„зӣёеҸҚж•°пјӣ
    • еӣ  $x_2 // x_3$пјҢд»Һ $x_2$ иҪҙжІҝ $z_2$ иҪҙиҪ¬еҲ° $x_3$ иҪҙзҡ„и§’еәҰдёә $\theta_3 = 0$пјӣ
    • д»Һ $z_2$ иҪҙжІҝ $x_3$ иҪҙеҲ° $z_3$ иҪҙзҡ„и·қзҰ»дёә $a_3$пјҢеҚі иҝһжқҶ $3$ з«–зӣҙж®өзҡ„й•ҝеәҰпјӣ
    • еӣ  $z_2 // z_3$пјҢд»Һ $z_2$ иҪҙжІҝ $x_3$ иҪҙиҪ¬еҲ° $z_3$ иҪҙзҡ„и§’еәҰдёә $\alpha_3 = 0$.
  4. $j = 4$ ж—¶пјҢ
    • д»Һ $x_3$ иҪҙжІҝ $z_3$ иҪҙеҲ° $x_4$ иҪҙзҡ„и·қзҰ»дёә $d_4$пјҢеҚіиҝһжқҶ $4$ й•ҝеәҰпјӣ
    • еӣ  $x_3 \perp x_4$ дё”дёҚе…ұйқўпјҢд»Һ $x_3$ иҪҙжІҝ $z_3$ иҪҙиҪ¬еҲ° $x_4$ иҪҙзҡ„и§’еәҰдёә $\theta_4 = -\pi/2$пјӣ
    • еӣ  $z_3 \perp z_4$ дё”е…ұйқўпјҢд»Һ $z_3$ иҪҙжІҝ $x_4$ иҪҙеҲ° $z_4$ иҪҙзҡ„и·қзҰ»дёә $a_4 = 0$пјӣ
    • д»Һ $z_3$ иҪҙжІҝ $x_4$ иҪҙиҪ¬еҲ° $z_4$ иҪҙзҡ„и§’еәҰдёә $\alpha_4 = -\pi/2$.
  5. $j = 5$ ж—¶пјҢ
    • д»Һ $x_4$ иҪҙжІҝ $z_4$ иҪҙеҲ° $x_5$ иҪҙзҡ„и·қзҰ»дёә $d_5$пјҢеҚіиҝһжқҶ $5$ й•ҝеәҰпјӣ
    • еӣ  $x_4 // x_5$пјҢд»Һ $x_4$ иҪҙжІҝ $z_4$ иҪҙиҪ¬еҲ° $x_5$ иҪҙзҡ„и§’еәҰдёә $\theta_5 = 0$пјӣ
    • еӣ  $z_4 \perp z_5$ дё”е…ұйқўпјҢд»Һ $z_4$ иҪҙжІҝ $x_5$ иҪҙеҲ° $z_5$ иҪҙзҡ„и·қзҰ»дёә $a_5 = 0$пјӣ
    • д»Һ $z_4$ иҪҙжІҝ $x_5$ иҪҙиҪ¬еҲ° $z_5$ иҪҙзҡ„и§’еәҰдёә $\alpha_5 = \pi/2$.
  6. $j = 6$ ж—¶пјҢ
    • д»Һ $x_5$ иҪҙжІҝ $z_5$ иҪҙеҲ° $x_6$ иҪҙзҡ„и·қзҰ»дёә $d_6$пјҢеҚіиҝһжқҶ $6$ й•ҝеәҰпјӣ
    • еӣ  $x_5 // x_6$пјҢд»Һ $x_5$ иҪҙжІҝ $z_5$ иҪҙиҪ¬еҲ° $x_6$ иҪҙзҡ„и§’еәҰдёә $\theta_6 = 0$пјӣ
    • еӣ  $z_5$ дёҺ $z_6$ йҮҚеҗҲпјҢд»Һ $z_5$ иҪҙжІҝ $x_6$ иҪҙеҲ° $z_6$ иҪҙзҡ„и·қзҰ»дёә $a_6 = 0$пјӣ

5 з»“жһңеҜ№жҜ”

з»ҷеҮә PeterCokeе·Ҙе…·з®ұдёӯUR10зҡ„ DH еҸӮж•°е’ҢдёҠиҝ°е»әз«Ӣзҡ„ DH еҸӮж•°д»ҝзңҹз»“жһңпјҡ

дёҠиҝ°з»“жһңзҡ„е·Ұдҫ§дёәе·Ҙе…·з®ұзҡ„д»ҝзңҹз»“жһңпјҢеҸідҫ§дёәжң¬ж–Үе»әз«Ӣзҡ„ DH еҸӮж•°д»ҝзңҹз»“жһңгҖӮвҖқoffsetвҖқеҚіиЎЁзӨәе…іиҠӮи§’зҡ„еҒҸиҪ¬пјҢеҜ№еә”дәҺ DH дёӯзҡ„ $\theta$гҖӮ

PeterCokeзҡ„е·Ҙе…·з®ұжҳҜдёҚеҢ…еҗ« UR10 зҡ„ STL жЁЎеһӢзҡ„пјҢARTE е·Ҙе…·з®ұдёӯеҢ…еҗ«дәҶиҜҘжЁЎеһӢпјҢйңҖиҰҒе°Ҷе®ғеҜје…ҘпјҢеҸҜд»ҘеҸӮиҖғ [5]гҖӮSTL жЁЎеһӢеҝ…йЎ»еҢ…еҗ«жүҖжңүиҝһжқҶй•ҝеәҰдҝЎжҒҜжүҚиғҪжӯЈеёёжҳҫзӨәгҖӮ

6 жҖқиҖғ

иҮіжӯӨпјҢиҝҳжңүдёӨдёӘе°Ҹе°Ҹзҡ„з–‘й—®дҫӣеӨ§е®¶жҖқиҖғпјҡ

  1. еҜ№дәҺйӣ¶дҪҚеңЁж°ҙе№іж–№еҗ‘зҡ„жғ…еҶөпјҢеҢ…еҗ«е…ЁйғЁеҸӮж•°зҡ„ DH еҸӮж•°жҳҜеӨҡе°‘пјҹ
  2. йӣ¶дҪҚеңЁж°ҙе№іж–№еҗ‘ж—¶пјҢиҝһжқҶ $1$ зҡ„й•ҝеәҰе’ҢиҝһжқҶ $3$ ж°ҙе№іж–№еҗ‘зҡ„й•ҝеәҰжҳҜеҗҰиў«иҖғиҷ‘еңЁ DH еҸӮж•°еҶ…дјҡеҪұе“ҚжңҖеҗҺзҡ„и®Ўз®—з»“жһңеҗ—пјҹ
